# OdenseNLP Website

GitHub Pages website for the OdenseNLP research group (SDU, Odense, Denmark).

## Quick Start

1. Edit content in `_posts/`, `_members/`, and `_data/`.
2. Adjust text pages (`index.md`, `members.md`, `publications.md`, etc.) if needed.
3. Replace placeholder logo and member images in `assets/images/`.
4. Push to the default branch used by GitHub Pages.
